UI Constraints — ChatList
1) Font: Inter (or system), body 16px, spacing 12px vertical between items.
2) Avatar size: 40px. Message preview truncated to 2 lines.
3) Unread badge: circular, red, top-right of list item.

UI Constraints — ChatRoom
1) Message bubble max width: 78% of screen width.
2) Timestamp small, 12px, muted color.
3) Input bar fixed above keyboard; send button next to input.

UI Constraints — Onboarding
1) Primary action (Generate Keys) bottom-centered, full width.
2) Show public key as small QR placeholder and copy button.
3) Minimal explanatory text (max 2 lines).
